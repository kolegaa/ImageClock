ImageClock Usage Guide

Welcome to ImageClock!

This web app returns the current time as an image. Use the /time endpoint with the following query parameters:

  • font: The font name (e.g., Arial, Pixel).
  • fontURL: A URL to a custom font file (e.g., .woff2).
  • tz: The time zone (e.g., America/New_York).
  • fontSize: The font size in pixels (default: 40).
  • color: The text color in RGB hex format (e.g., #FF0000).
  • bgColor: The background color in RGB hex format or transparent (default: transparent).

Example Usage:

  • /time?font=Arial&tz=America/New_York&fontSize=50&color=#FF0000&bgColor=#FFFFFF
  • /time?font=Pixel&tz=UTC&fontSize=30&color=#0000FF&bgColor=transparent
  • /time?fontURL=https://example.com/custom_font.woff2&tz=Europe/London

Enjoy using ImageClock!
